# My Ip Address

Show client IP address

- Browser: /index.html
- API: GET /refresh

## Project Setup

```sh
make install
# or
npm install
```

### Compile and Hot-Reload for Development

```sh
make dev
# or
npm run dev
```

### Lint with [ESLint](https://eslint.org/)

```sh
make lint
# or
npm run lint
```

### Format with [Prettier](https://prettier.org/)

```sh
make format
# or
npm run format
```

### Cloudflare Wrangler Type Generation

```sh
make typegen
# or
npm run cf-typegen
```

### Cloudflare Wrangler Deploy

```sh
make deploy
# or
npm run deploy
```
